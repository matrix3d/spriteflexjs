This is the documentation for the Royale compiler and its supporting libraries.

The code is organized into six top-level packages:

org.apache.royale.abc Library for manipulating ActionScript byte code
org.apache.royale.compiler The Royale compiler
org.apache.royale.swc Library for reading and writing SWC files
org.apache.royale.swf Library for reading and writing SWF files
org.apache.royale.utils Library containing miscellaneous utilities

For more information, read the description of each of these packages.
